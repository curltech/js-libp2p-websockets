▸ filter(multiaddrs)
src/index.js

Takes a list of Multiaddrs and returns only valid Websockets addresses. By default, in a browser environment only DNS+WSS multiaddr is accepted, while in a Node.js environment DNS+{WS, WSS} multiaddrs are accepted.
